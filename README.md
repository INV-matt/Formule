#### Asse del segmento
$$(x-x_A)^2+(y-y_A)^2=(x-x_B)^2+(y-y_B)^2$$
$$2(x_B-x_A)+2(y_B-y_A)+x_A^2+y_A^2-x_B^2-y_B^2=0$$
